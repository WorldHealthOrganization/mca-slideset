The BMat 2019 model follows the 2015 specification, as available here
https://www.who.int/reproductivehealth/publications/monitoring/maternal-mortality-2015/en/
and described here

Alkema L, Zhang S, Chou D, Gemmill A, Moller AB, Ma Fat D, et al. A Bayesian approach to the global estimation of maternal mortality. Ann Appl Stat. 2017;11(3):1245-74. doi:10.1214/16-aoas1014.

Updates are described in the UN MMEIG 2019 report, with technical details available here

Peterson E, Chou D, Gemmill A, Moller AB, Say L, Alkema L. (2019) Estimating maternal mortality using vital registration data: a Bayesian hierarchical bivariate random walk model to estimate sensitivity and specificity of reporting. ArXiv.

The JAGS model file corresponding to the BMat 2019 model is included in this zip-file, as is the input data file. 

Main.R can be used to fit the model.

The outputs, in the form of posterior samples for the number of maternal deaths for each country-year, are available here
https://www.who.int/reproductivehealth/publications/monitoring/maternal-mortality-2019/en/
